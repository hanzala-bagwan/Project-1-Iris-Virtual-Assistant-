# Python Voice Assistant:- Iris

A Python voice assistant that can perform basic tasks using voice commands.

## Features
- Voice recognition
- Text-to-speech responses
- Open websites (Google, YouTube, LinkedIn, Instagram)
- Play music using voice commands

## Libraries Used
- speech_recognition
- pyttsx3
- webbrowser
- time

## How It Works
1. The assistant listens for a wake word.
2. After activation, it processes voice commands.
3. Commands are matched with actions like opening websites or playing music.

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install -r requirements.txt
